# OBS Timestamp Logger (Lua Version)

OBSで配信または録画を開始した際に自動的にログファイルを作成し、ホットキーでタイムスタンプを記録するスクリプトです。

## 機能

- **自動ファイル生成**: 配信または録画開始時に `YYYY-MM-DD_HH-MM.txt` という名前のファイルを指定フォルダに作成します。
- **タイムスタンプ記録**: 設定したホットキーを押すと、現在のファイルに「配信開始からの経過時間（`HH:MM:SS`）」を追記します。
- **再接続対応**: 配信終了から**設定した時間（デフォルト30分）以内**に再開した場合、確認ポップアップが表示されます。「はい」を選ぶと続きから記録でき、「いいえ」を選ぶと新規作成されます。

## インストール手順

1. **スクリプトの追加**:
   - OBSのメニューから `ツール` -> `スクリプト` を選択します。
   - `スクリプト` タブの左下にある `+` ボタンを押します。
   - このフォルダにある `timestamp_logger.lua` を選択します。

## 設定方法

1. **保存場所の設定**:
   - スクリプト画面の右側に設定が表示されます。
   - 「保存先フォルダ」の行で、ログファイルを保存したいフォルダを選択してください。

2. **ホットキーの設定**:
   - OBSの `設定` -> `ホットキー` より選択してください。
   - 項目名: `Timestamp Log: Mark Time`

## 使い方

1. **配信** または **録画** を開始します。
   - 設定したフォルダに、現在日時のファイル（例：`2026-02-04_14-30.txt`）が自動作成されます。
2. 重要なシーンなどで設定した **ホットキー** を押します。
   - そのファイルに `14:35:10` のような時間が追記されます。
3. 配信・録画を終了すると、ファイルの記録も終了します。
